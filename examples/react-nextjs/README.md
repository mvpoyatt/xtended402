# React + Next.js Frontend

Part of the xtended402 e-commerce example.

## Setup

See [/examples/README.md](../README.md) for complete setup instructions and how this frontend works with the Go backend.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

**Note:** The Go backend must be running on `http://localhost:8080` for this to work.
